# jd_maotai
京东 茅台 抢购软件

本软件仅用于娱乐，禁止用于任何违法的事。


本软件需要打开京东APP，扫码登录，运行之前需要配置cookie。
关于cookie的提取方法，可以关注公众号，回复 “京东茅台” 获取详细信息。

# 怎么判断软件是否成功运行
1. 预约成功。
  
目前京东茅台只能通过APP才能预约成功，运行软件之后，打开京东APP，发现商品预约成功。

2. 抢购

整个抢购过程中，如果没有出现异常，有以下类似信息打印，那么恭喜你，你可以用这个软件抢购了。
